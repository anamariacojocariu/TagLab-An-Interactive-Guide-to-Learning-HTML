Definirea stilurilor pentru liste

Pentru a defini stilurile pentru liste în CSS, putem modifica diverse proprietati care afecteaza atat listele ordonate (ol) cat si listele neordonate (ul). Iata cateva exemple de cum putem stiliza listele:


Pentru inceput, vom construi o lista ordonata, nestilizata, pentru a putea observa diferentele ulterior:

  1. Primul element
  2. Al doilea element
  3. Al treilea element
  4. Al patrulea element
  5. Al cincilea element

Pentru primul exemplu de stilizare se va utiliza proprietatea: list-style. Aceasta seteaza toate proprietatile pentru o lista.

  1. Primul element
  2. Al doilea element
  3. Al treilea element
  4. Al patrulea element
  5. Al cincilea element

Pentru al doilea exemplu de stilizare se va utiliza proprietatea: list-style-type. Aceasta indica tipul de marcator folosit pentru elementele din lista.

Square:

  1. Primul element
  2. Al doilea element
  3. Al treilea element
  4. Al patrulea element
  5. Al cincilea element

Circle:

  1. Primul element
  2. Al doilea element
  3. Al treilea element
  4. Al patrulea element
  5. Al cincilea element

Lower greek letters:

  1. Primul element
  2. Al doilea element
  3. Al treilea element
  4. Al patrulea element
  5. Al cincilea element

etc.
